JavaScript練習

ウェブサイトを作成するためには、次の三つの道具が必要です。

  1. コンピュータ(Mac / Linux / Windows など)
  2. エディタ (Pulsar / Zed / VS Codeなど)
  3. ブラウザ (Firefox, Safariなど)

ウェブサイトの作成には、次の三つの言語が良く使われます。

  • HTMLは、「文書構造」の為の言語です。
  • CSSは、「意匠・装飾」の為の言語です。
  • JavaScriptは、
    「相互作用・動き・制御」の為の言語です。
